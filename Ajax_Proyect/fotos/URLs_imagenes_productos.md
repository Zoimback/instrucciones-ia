# URLs de Imágenes de Productos Ajax - Descarga Manual

Este documento contiene las URLs de imágenes oficiales de Ajax Systems para todos los productos del catálogo.

## Instrucciones de Descarga

1. Copia la URL de la imagen
2. Abre en navegador o usa `wget` o `curl`
3. Guarda en la carpeta correspondiente:
   - `/fotos/blancos/` - Productos blancos (sufijo -W o "Blanco")
   - `/fotos/negros/` - Productos negros (sufijo -B o "Negro")
