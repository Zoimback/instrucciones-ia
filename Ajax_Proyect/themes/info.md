## Notas de Implementación

### Full Site Editing (FSE)
Este sitio utiliza tecnología FSE moderna de WordPress, lo que significa:
- Las plantillas se editan visualmente en el editor de bloques
- Usa `theme.json` para configuración global
- Los estilos se gestionan mediante bloques
- No usa los archivos PHP tradicionales de plantilla

### Archivos Importantes
- `theme.json` - Configuración global de tema
- `style.css` - Estilos principales
- `/patterns/` - Patrones de bloque personalizados
- `/parts/` - Partes de plantilla (header, footer, etc.)
- `/templates/` - Plantillas de página

### Customizaciones
Para personalizar el tema, se recomienda:
1. Usar el Editor de Sitio (Apariencia → Editor)
2. Modificar `theme.json` para tokens de diseño
3. Crear patrones personalizados
4. No modificar archivos del tema padre directamente
